# Extensibility assessment CSV reference

The Extensibility component writes:

```text
classicextensibilities.csv
classicusercustomactions.csv
```

Both files use `ScanId`, `SiteUrl`, and `WebUrl` to identify the affected web.

## `classicextensibilities.csv`

This file contains one row per web with a detected custom master page, alternate CSS, or incompatible user custom action.

| Column | Description |
| --- | --- |
| `ScanId` | Assessment identifier. |
| `SiteUrl` | Absolute site-collection URL. |
| `WebUrl` | Site-collection-relative web URL. `/` represents the root web. |
| `UsesCustomMasterPage` | Indicates that `MasterPage` or `CustomMasterPage` isn't in the tool's out-of-box exclusion list. |
| `MasterPage` | Custom value from the web's system master-page property. |
| `CustomMasterPage` | Custom value from the web's site master-page property. |
| `UsesCustomCSS` | Indicates that alternate CSS is configured. |
| `AlternateCSS` | Alternate CSS URL. |
| `UsesCustomTheme` | Compatibility field. The current Extensibility component doesn't populate custom-theme findings. |
| `UsesUserCustomAction` | Indicates that at least one incompatible user custom action was stored for the web. |
| `HasSharePointAddIns` | Compatibility field. The current Extensibility component doesn't populate SharePoint Add-In findings. |
| `RemediationCode` | Comma-separated Extensibility remediation codes found for the web. |

Out-of-box master pages are excluded. A web with no detected Extensibility finding doesn't receive a row.

## `classicusercustomactions.csv`

This file contains incompatible Site-, Web-, or List-scoped user custom actions.

| Column | Description |
| --- | --- |
| `ScanId` | Assessment identifier. |
| `SiteUrl` | Absolute site-collection URL. |
| `WebUrl` | Site-collection-relative web URL. `/` represents the root web. |
| `Id` | User custom action identifier. |
| `Title` | Display title. |
| `Name` | Internal name. |
| `Location` | Custom action location. |
| `RegistrationType` | Registration type associated with `RegistrationId`. |
| `RegistrationId` | Registration target. |
| `CommandAction` | JavaScript command extracted from a command UI handler when detected. |
| `CommandUIExtension` | Command UI extension XML. |
| `Description` | Description. |
| `ScriptBlock` | Inline script captured for an incompatible location. |
| `ScriptSrc` | External script source captured for an incompatible location. |
| `Url` | Configured action URL. |
| `Problem` | Detected problem, such as `InvalidLocation` or `JavaScriptEmbedded`. |
| `ListId` | List identifier for a List-scoped action. |
| `ListUrl` | List URL for a List-scoped action. |
| `ListTitle` | List title for a List-scoped action. |
| `RemediationCode` | `CE6`, `CE8`, or `CE9`, depending on scope and detected incompatibility. |

The row key is `ScanId`, `SiteUrl`, `WebUrl`, and `Id`.

Only actions with a populated `Problem` are stored. Compatible actions don't appear in this file. For a List-scoped incompatible action, the current implementation assigns `CE6` after recording the detected problem.

Site-scoped actions are evaluated when the root web is scanned. Web-scoped and List-scoped actions are evaluated in their respective web.

## Interpret remediation codes

| Code | Meaning in this output |
| --- | --- |
| `CE3` | Custom master page. |
| `CE4` | Alternate CSS. |
| `CE6` | List-scoped incompatible user custom action. |
| `CE8` | JavaScript embedded in a command UI handler. |
| `CE9` | Other incompatible custom-action location. |

`CE5` exists in the model for custom themes but isn't populated by the current component.

For execution guidance, see [Assess classic SharePoint extensibility](assessment-tool-extensibility.md).
