Encodes a string to fit within JavaScript code. This element is used to
render text from the server that must be interpreted as a quoted string.

The **ScriptQuote** element performs the
following conversions on its body text:

-   converts " to \\"

-   converts \\ to \\\\

-   converts CR to \\r\\CR (where CR is character code 13)

For example, the code <span class="code">\<ScriptQuote\>This is a
test\</ScriptQuote\></span> returns "This is a test" because <span
class="keyword">NotAddingQuote</span> is not set to <span
class="keyword">TRUE</span>. The code <span class="code">\<ScriptQuote
NotAddingQuote="TRUE"\>This is a test\</ScriptQuote\></span>, however,
returns This is a test, without quotation marks, because <span
class="keyword">NotAddingQuote</span> is set to <span
class="keyword">TRUE</span>. Note that in the second case, the string is
not surrounded by quotation marks.
