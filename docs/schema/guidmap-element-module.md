# GUIDMap element (Module)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies the GUID map for a web part that was serialized to binary data.

## Definition

```XML
<GUIDMap />
```
