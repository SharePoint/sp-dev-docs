# Data Element (Event)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Contains a string that is used to pass parameters to the receiver.

## Definition

```XML
<Data>
</Data>
```
