Specifies the friendly name of the SharePoint Add-in that appears in the
user interface (UI). The name cannot contain more than 256 characters.

**Last modified:** September 17, 2015

**Applies to**: apps for SharePoint | SharePoint Foundation 2013 |
SharePoint Server 2013*
