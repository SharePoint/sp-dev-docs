# Field element (Fields element) 

(SchemaSource element) (dataSourceType complexType) (Action4)

**Applies to**: SharePoint Server 2013

Specifies a field in which to display data obtained from the parent data source.

## Usage

```XML
    <SchemaSource>
        <Fields>
            <Field />
        </Fields>
    </SchemaSource>
```
