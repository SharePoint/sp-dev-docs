# Property element (Delegate Control)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Contains a property value for a delegate control.

## Definition

```XML
<Property
  Name = "Text">
</Property>
```
