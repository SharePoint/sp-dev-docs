# CommandUIDefinitions element (CommandUIExtensionType complexType) 

(AppHostWebFeatures)

> [!NOTE] 
> The string `app` appears as part of or all of some element, attribute, and file names because SharePoint Add-ins were originally called "apps for SharePoint." To ensure backward compatibility, the schemas have not been changed. 

## Element information

| Element Type | Namespace | Schema File |
|--------------|-----------|-------------|
| CommandUIDefinitionsType | `http://schemas.microsoft.com/sharepoint/` | apphostwebfeatures.xsd |

## Definition

```XML
    <xs:element name="CommandUIDefinitions" type="CommandUIDefinitionsType" minOccurs="0" maxOccurs="1"></xs:element>
```
