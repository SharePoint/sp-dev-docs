# Parameters Element (Feature)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies the list of parameters to be applied during a Feature upgrade.

## Definition

```XML
<Parameters> 
  <Parameter> </Parameter>
</Parameters>
```
