# TemplateFiles Element (Solution)

 
  
 **Applies to:** SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013
  
Specifies the template files to include in the solution.
  
```
<TemplateFiles
  <TemplateFile
    Location="Text"/>
    ...
</TemplateFiles>
```
