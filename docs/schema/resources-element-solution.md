# Resources Element (Solution)

 
  
 **Applies to:** SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013
  
Specifies the resources to be deployed to an existing Feature folder.
  
```
<Resources
  <Resource
    Location="Text"/>
    ...
</Resources>
```
