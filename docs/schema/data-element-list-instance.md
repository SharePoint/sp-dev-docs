# Data Element (List Instance)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies data to include by default in lists created through a list definition.

## Definition

```XML
<Data>
</Data>
```
