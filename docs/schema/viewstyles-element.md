# ViewStyles element

**Applies to:** SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013
  
Contains the collection of view styles within a ViewStyles.xml file. 
  
```XML

</ViewStyles>
```
