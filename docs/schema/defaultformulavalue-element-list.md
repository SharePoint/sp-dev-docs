# DefaultFormulaValue element (List)

**Applies to:** SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013
  
Contains the default value for a field that has a formula. 
  
```XML
<DefaultFormulaValue>
</DefaultFormulaValue>
```
