# WorkflowInfo element (Action4)

**Applies to**: SharePoint Server 2013

The root element of the Actions schema. This element must be included in any .ACTIONS file that is installed on the server.

## Usage

```XML
    <WorkflowInfo>
        <Conditions />
        <Actions />
        <CompositeSteps />
        <VariableTypes />
        <AssemblyRedirects />
        <Coercions />
        <Events />
        <Flows />
        <Blocks />
    </WorkflowInfo>
```
