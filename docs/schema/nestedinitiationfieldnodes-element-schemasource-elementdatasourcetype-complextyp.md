# NestedInitiationFieldNodes element 

(SchemaSource element) (dataSourceType complexType) (Action4)

**Applies to**: SharePoint Server 2013

Specifies that fields are retrieved from the schema of the data used to begin the workflow.

## Usage

```XML
    <SchemaSource>
        <NestedInitiationFieldNodes />
    </SchemaSource>
```
