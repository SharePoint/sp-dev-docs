# Property element (Feature)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Defines the default value for a feature property.

## Definition

```XML
<Property
  Key = "Text"
  Value = "Text">
</Property>
```
