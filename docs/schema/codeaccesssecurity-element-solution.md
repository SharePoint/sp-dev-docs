# CodeAccessSecurity Element (Solution)

 
  
 **Applies to:** SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013
  
Specifies the code access security for the solution.
  
```
<CodeAccessSecurity>
  <PolicyItem>
  ...
  </PolicyItem>
</CodeAccessSecurity>
```
