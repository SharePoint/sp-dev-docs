# DataSources Element (WorkflowInfo)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies the data sources for the action or composite step.

## Definition

```XML
<DataSources>
    <DataSource>
    </DataSource>
</DataSources>
```
