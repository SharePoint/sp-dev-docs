# DataSources Element (WorkflowActions)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies the data sources for the action.

## Definition

```XML
<DataSources>
    <DataSource>
    </DataSource>
</DataSources>
```
