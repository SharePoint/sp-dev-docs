# HashtableSource Element (WorkflowInfo)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies that the evaluation value is retrieved from a hash table.

## Definition

```XML
<Evaluation>
    <HashtableSource>
    </HashtableSource>
</Evaluation>
```
