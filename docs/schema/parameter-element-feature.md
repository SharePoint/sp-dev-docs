# Parameter Element (Feature)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies the parameter to be applied during a Feature upgrade.

## Definition

```XML
<Parameter 
  Name = string>
</Parameter>
```
