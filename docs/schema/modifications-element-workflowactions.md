# Modifications Element (WorkflowActions)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 |SharePoint Online | SharePoint Server 2013

Specifies the modifications that are associated with the action.

## Definition

```XML
<Actions>
    <Action>
        <Modifications>
            <Modification>
            </Modification>
        </Modifications >
    </Action>
</Actions>
```
