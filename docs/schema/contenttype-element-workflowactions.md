# ContentType Element (WorkflowActions)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies a content type from which to retrieve fields.

## Definition

```XML
<SchemaSource>
    <ContentType>
    </ContentType>
</SchemaSource>
```
