# Fields element (WorkflowActions)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies fields for displaying data from the parent data source.

## Definition

```XML
<SchemaSource>
    <Fields>
        <Field>
        </Field>
    </Fields>
</SchemaSource>
```
