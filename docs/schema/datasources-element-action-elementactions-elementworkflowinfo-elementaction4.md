# DataSources element (Action element) 

(Actions element) (WorkflowInfo element) (Action4)

**Applies to**: SharePoint Server 2013

Specifies the data sources for an action.

## Usage

```XML
    <Action>
        <DataSources>
            <DataSource />
        </DataSources>
    </Action>
```
