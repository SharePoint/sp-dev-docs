The URL you specify must be server-relative.

If you do not specify a custom status page, SharePoint Foundation uses
the default status page, located at <span
class="code">\_layouts/WrkStat.aspx</span>.


------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

workflow.
