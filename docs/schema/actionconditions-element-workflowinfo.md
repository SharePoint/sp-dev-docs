# ActionConditions element (WorkflowInfo)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies conditional logic for the action.

## Definition

```XML
    <Actions>
        <Action>
            <ActionVariables>
            </ActionVariables>
            <ActionBody>
            </ActionBody>
            <ActionConditions>
            </ActionConditions>
        </Action>
    </Actions>
```
