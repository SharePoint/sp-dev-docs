# MapFile element (Feature)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies a mapping between files during a Feature upgrade.

## Definition

```XML
<MapFile 
  FromPath = string
  ToPath = string />
```
