# Dictionary element (WorkflowInfo)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies a dictionary of name-value pairs from which to retrieve fields.

## Definition

```XML
<SchemaSource>
    <Dictionary>
    </Dictionary>
</SchemaSource>
```
