# FormulaDisplayNames element (List)

**Applies to:** SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013
  
Contains the display names of formulas used within a list field. 
  
```XML
<FormulaDisplayNames>
</FormulaDisplayNames>
```
