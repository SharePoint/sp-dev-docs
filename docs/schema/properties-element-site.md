# Properties Element (Site)

 
  
 **Applies to:** SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013
  
Specifies custom property values to implement within the Feature. 
  
```
<Properties>
    <Property
      Key = "Text"
      Value = "Text">
    </Property>
</Properties>
```
