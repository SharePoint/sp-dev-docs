# WebPartTransformer Element (Module)

**Applies to**: SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013

Specifies the transformer for the web part.

## Definition

```XML
<WebPartTransformer 
   Assembly = string 
   Namespace = string />
```
