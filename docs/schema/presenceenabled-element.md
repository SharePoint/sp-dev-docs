# PresenceEnabled Element

 
  
 **Applies to:** SharePoint 2016 | SharePoint Foundation 2013 | SharePoint Online | SharePoint Server 2013
  
Contains a Boolean value that specifies whether presence is enabled.
  
```
<PresenceEnabled>
</PresenceEnabled>
```
